# 功能演示说明

## 🎯 系统架构

```
用户输入 → 意图识别 → Agent分发 → 流式输出
           ↓
    [天气/数学/知识/代码/通用]
```

## ✨ 核心功能

### 1. 左右对话气泡界面
- **用户消息**：显示在右侧，紫色渐变背景
- **AI 回复**：显示在左侧，白色背景带边框
- **头像标识**：用户 👤 vs AI 🤖
- **时间戳**：每条消息显示发送时间

### 2. 流式输出（Streaming）
- **实时显示**：AI 回答逐字显示，类似打字机效果
- **流畅体验**：无需等待完整响应
- **视觉反馈**：显示"正在输入"动画
- **技术实现**：使用 Server-Sent Events (SSE)

### 3. 意图识别展示
每条 AI 消息下方显示：
- 🌤️ 天气助手 · 天气查询
- 🔢 数学助手 · 数学计算
- 📚 知识助手 · 知识问答
- 💻 代码助手 · 代码助手
- 💬 通用助手 · 通用对话

### 4. 侧边栏快捷示例
点击即可快速发送：
- 天气查询示例
- 数学计算示例
- 知识问答示例
- 代码帮助示例
- 通用对话示例

## 🚀 使用流程

1. **打开浏览器**：访问 http://localhost:8080
2. **输入问题**：在底部输入框输入你的问题
3. **发送消息**：按 Enter 或点击发送按钮
4. **查看结果**：
   - 用户消息立即显示在右侧
   - AI 开始思考（显示加载动画）
   - 意图识别结果显示
   - AI 回答逐字显示在左侧
   - 显示处理的 Agent 信息

## 💡 交互特性

### 键盘快捷键

- 输入框高度随内容自动调整

### 视觉反馈
- 发送按钮悬停效果
- 示例问题悬停动画
- 在线状态脉冲动画
- 消息淡入动画

## 🎨 界面元素

### 顶部标题栏
- 系统名称：多 Agent 智能对话
- 在线状态指示器（绿色脉冲点）

### 左侧边栏
- 系统 Logo 和名称

### 中间聊天区

  - 头像
  - 消息内容

### 底部输入区

- 发送按钮（纸飞机图标）
- 圆角边框设计

## 🔧 技术亮点

### 后端
- **Spring Boot 4.0.6**：现代化 Web 框架
- **LangChain4j 0.34.0**：AI 应用开发框架
- **SSE (Server-Sent Events)**：流式数据传输
- **异步处理**：CompletableFuture 提高性能

### 前端
- **原生 JavaScript**：无依赖，轻量级
- **Fetch API + ReadableStream**：流式数据接收
- **CSS3 动画**：流畅的视觉效果
- **Flexbox 布局**：响应式设计

### AI 集成
- **阿里云百炼**：通义千问大模型
- **OpenAI 兼容接口**：标准化调用
- **多 Agent 架构**：专业化分工
- **智能路由**：基于意图的分发

## 📊 性能优化

1. **流式传输**：减少首字延迟
2. **异步处理**：不阻塞主线程
3. **按需渲染**：只更新变化的 DOM
4. **内存管理**：及时清理无用元素
5. **防抖处理**：避免频繁请求
